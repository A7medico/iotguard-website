Features

🧠

Hybrid AI Detection

Combines supervised LightGBM classification with unsupervised IsolationForest anomaly detection for maximum coverage.

🔍

SHAP Explainability

Every detection comes with human-readable explanations showing which features contributed most to the decision.

⚡

Real-time Processing

Sub-millisecond inference with streaming pipeline. Scores network windows as they arrive, no batch delays.

🛡️

Automated Response

Configurable blocking with severity levels (SOFT, HARD, KILL). Supports Windows netsh and Linux nftables.

📊

Live Dashboard

Professional monitoring interface with real-time charts, attack distribution, event filtering, and config controls.

🔄

Adaptive Thresholds

Self-adjusting detection thresholds based on recent traffic patterns. Reduces false positives in dynamic environments.

Architecture
Modular pipeline design
Five-stage pipeline from raw network traffic to automated threat response.
1

Data Input

Suricata EVE logs or simulated streams → feature extraction

2

Training

LightGBM + IsolationForest trained on 2.1M+ IoT flows

3

Inference

Decision loop scores windows, classifies attacks with SHAP

4

Response

Alerts, firewall blocking, and audit logging

5

Dashboard

REST API + WebSocket for real-time monitoring
